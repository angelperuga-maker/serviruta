Configura tu API key de Anthropic
Los agentes usan Claude Sonnet 4.5 real. Necesitas configurar tu API key de Anthropic en Supabase (una sola vez):
  1. Ve a console.anthropic.com/settings/keys y crea una API key
  2. En Supabase: Edge Functions → chat-agente → Secrets
  3. Añade una variable: ANTHROPIC_API_KEY con tu key
  4. Recarga esta página

🔌 Integración con WhatsApp Business

Paso 1: Conecta WhatsApp Business Cloud API (Meta) o Whapi.cloud

Paso 2: Configura webhook en Make.com que reciba mensajes y llame al endpoint de agentes de Supabase.

Paso 3: Make.com decide qué agente responde según primera clasificación con Rita.

Paso 4: El agente responde con la API de Claude usando el prompt correspondiente.

Paso 5: La respuesta se envía por WhatsApp de vuelta y la conversación se guarda en Supabase.

💡 Flujo simplificado con Make.com:
1. [WhatsApp] → Cliente escribe
2. [Webhook Make] → Recibe mensaje  
3. [Supabase RPC] → Busca o crea conversación
4. [Claude API] → Genera respuesta con prompt del agente
5. [Supabase] → Guarda mensaje en conversaciones_agentes
6. [WhatsApp] → Envía respuesta al cliente
